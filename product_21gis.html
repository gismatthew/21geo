Supporting the Web Map Service (WMS) standard, 21st-WebGIS can create maps in a variety of output image format. Moreover, 21st-WebGIS conforms to the Web Feature Service (WFS) standard, which allows the actual sharing and editing of the spatial data, so users can edit the the online feature layer. 21st-WebGIS also support Tile Map Service (TMS) to accelerate online maping rendering. On the Client side, 21st-WebGIS support mapping library OpenLayers, LeafLet, Google Map API, making map generation quick and easy.

With great flexibility in map creation, data sharing and online editing, 21st-WebGIS platform can be easily extended to other Web platform developments, such as city management, environmental assessment etc.

Based on 21st-WebGIS platform, we developed Saskatoon Information Hub (www.nicesask.ca), Immigration WebGIS.
